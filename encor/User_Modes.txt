====== User Modes ======
Created Tuesday 26 September 2023

User mode: '>'
Privileged User mode: '#'
Global or Global Configuration mode: 'device (config)#'
	its global because you haven't selected an interface or router or anything

0 is just logout, enable, disable, help, exit

1 lets you ping and traceroute and show int (not show RUN int)
